Overview

In this assignment, you must work with countries. The Country record class represents a country, and it has the following components:

  • String code - Country code (e.g., "HU")
  • String name - Country name (e.g., "Hungary")
  • String capital - Capital city
  • Region region - Geographic region
  • long population - Population count
  • BigDecimal area - Country area in square kilometers
  • Map<String, String> translations - Country name translations
  • List<ZoneId> timezones - Time zones in the country
  • boolean independent - Whether the country is independent

Under the src/main/resources/countries directory, you can find a JSON file named countries.json. It contains a list of countries from around the world.

The countries.CountriesInterface interface provides support to work with Country objects. Its default method getCountries() returns the list of Country objects from the repository.

The interface contains five abstract interface methods; you can see them in the source code.

Implementing the New Feature

JSON Export Feature

Now you will implement a fancy new feature that exports countries to JSON format using Gson.

Modifications to Make

  1. Add a new method to the CountriesInterface:

 * Exports all countries to a JSON string using Gson

  2. Implement this method in CountriesInterfaceImpl: 
    @Override
public String exportCountriesToJson() {

    return gson.toJson(getCountries());

  • Use proper filtering and mapping for different stream pipelines
